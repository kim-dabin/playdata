## JS Dom

- document : Dom의 최고 상위 객체, html 문서
- element : html문서에 사용되는 모든 태그, 요소
- attribute : 엘리먼트가 가진 속성
- node : 객체를 지칭, 요소, 속성, 텍스트 노드
